# frontend-sourcecode
This repository contains a source code of my website created using vscode for the Hackathon
Engictivity is a website designed to help engineering stdents , organize their daily academic activities, track progress, and build consistent study habits.
Our website Engictivity focuses on simplicity, usability and real-world student workflows.
1. Problem Statement
Students often struggle to manage mutiple academic resposibilities such as lectures,assignments,practice sessions,exams and self study. Exixsting tool are either too 
complex,generic or not tailored to academic routines.
Engictivity solves this problem by providing:
Clear daily planning,
Subject-wise academic organization,
Visual progress tracking,
A minimal and focused UI that is distraction free.
2. Tech Stack
HTML5-Structure and content,
CSS3-Styling,layout and responsiveness,
VS Code-Development environment,
3. Future Enhancements
Task creation and deletion,
Progress calculaton basedon completed task,
Dark mode,
User Authentication.
4. How to Use this website:
Navigation bar. Use the navigation links at the top to move between sections: Today,Academics,Progress and About.
5. Engictivity demonstrates how thoughtful UI design and clear problem understanding can create meaningful productivity tools.
